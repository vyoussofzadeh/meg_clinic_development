Dipole File Format

Brainstorm can import dipoles that are in the Neuromag .bdip format. Typically these will have been generated using the Neuromag Source Modelling module, xfit.  For more information about xfit and the bdip format see Neuromag Software Manuals.

 

To Import:

  • Select the study where you want to import the dipole file
  • Right click and select ‘Import xfit dipoles’

 

 

Subsets:

You will be prompted to specify the number of subsets used for dipole fitting. 

In most cases, this will be one.  If subsets are used, as defined in the text file $HOME/.meg_analysis/xfit_selections, then the number of subsets used needs to be input when prompted.

 

 

Visualizing dipoles on the brain

Dipoles can be defined as a surface and visualized on the cortical surface, 3D MRI, or with the MRI Viewer.

·         Select the dipole file. 

·         Right click and select the surface. You can use the functions in the dipole panel to interact with your displayed dipoles.

 

Loaded Dipoles:

A list of dipoles, grouped according to event (Group #1, Group #2, etc) and by channel subset, if used (Subset #1, Subset #2, etc).

·         Select the Group(s) to plot.  Clicking on a subset will select all Groups with that subset.  For example, selecting Subset #1 will select Group #1 (1), Group #2 (1), Group #3 (1), etc.

Display Options:

·         The Goodness Slider will show all selected dipole fits that had a goodness of fit measure greater than the value of the slider.

·         Checking the ‘Show max goodness of fit’ will show the maximum goodness of fit from the selected dipoles

·         Checking the ‘Show all time’ will plot all time points in the selected groups.  Un-checking this box shows only those fits from the selected groups at the time specified in the time window at the top of the panel.

·         When reviewing dipoles with the 3D images, the colors of the dipoles can be defined by the group, time and goodness of fit.  For example if ‘Show all Time’ and color according to ‘Time’ are both checked, the dipoles will be displayed with blue being the earliest fits propagating to red with the last fit in the group.
